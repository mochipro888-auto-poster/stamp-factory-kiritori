# 🏭 スタンプ工場『きりとり』

ChatGPTで作ったスタンプ画像を **切り抜き → 透過 → ZIPダウンロード** するWebアプリ。
誰でも簡単にLINEスタンプ商品化が可能。

**by [りーま@AI薬剤師｜すきぷら](https://twitter.com/leama_55555)**

---

## ✨ 主な機能

### 🖱️ 手動範囲選択（ハイブリッドUX）
- マウスドラッグで自由に範囲指定
- グリッド初期配置（2×2 / 3×3 / 4×4 / 4×6 / 4×8 / 5×8 等11種類）
- 各矩形を移動・リサイズ・削除可能
- アスペクト比ロック（出力サイズに合わせて自動調整）

### 🤖 AI自動検出
- 連結成分検出＋クラスタリングで全オブジェクトを自動配置
- ChatGPT生成画像に最適化
- セリフ文字も本体と統合

### 📐 25種類のサイズプリセット
- LINE スタンプ（370×320 / 240×240 / 96×74）
- X / Instagram / Threads / note / Brain / YouTube
- A4縦・横、汎用サイズ、カスタムサイズ

### 🛠️ 個別画像エディター
- プレビュー画像クリックで拡大編集
- 🧹 消しゴム（ブラシサイズ円カーソル）
- ⬛ 範囲削除
- 🤖 自動削除（中心から離れた小オブジェクトを一括削除）
- ↩️ 履歴30回・元に戻す

### 🎨 背景透過
- 白背景の自動透明化
- 閾値・滑らか度調整可能
- 透過PNG出力

### 📦 ZIP一括ダウンロード
- スタンプ画像（各サイズ）+ メイン画像（240×240）+ タブ画像（96×74）
- README.txt付き（LINE申請ガイド同梱）

---

## 🚀 使い方

1. ChatGPTでスタンプ画像を生成（4×4の16枚推奨）
2. このページにアップロード
3. 「4×4」or「🤖 AI自動検出」で初期配置
4. 必要なら各枠をドラッグして微調整
5. 「✂️ 切り抜きを実行」でプレビュー
6. 各画像をクリックして個別編集も可能
7. 「📥 ZIP一括ダウンロード」でPNG群を取得
8. LINEスタンプメーカーに提出！

## 🛠️ 技術スタック

| 要素 | 技術 |
|---|---|
| HTML/CSS | TailwindCSS（CDN） |
| JavaScript | Vanilla JS（フレームワーク不要） |
| 画像処理 | Canvas API |
| ZIP生成 | JSZip（CDN） |
| ホスティング | Cloudflare Pages |

完全クライアントサイドで動作。**画像はサーバーに送信されません**（プライバシー◎）。
