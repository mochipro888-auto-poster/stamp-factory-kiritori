```
▼ ① キャラ画像（必須）：1枚添付してください
```

### 2-2. 画像をアップロード

### 2-3. 範囲を自動配置

**おすすめ：「🤖 AI自動検出」ボタンをクリック**

→ 検出された枠が自動で配置されます。

うまく検出されなかったり、はみ出ていたら、**手動で微調整**：

### 2-4. ペンツールで線を描く（白服対策・必要なら）

白い服や白い装飾に黒い輪郭線がない場合、自動透過で内側まで消えてしまうことがあります。

その場合は **「✏️ ペン」** ボタンをクリックしてペンモードON：

- 🖌️ **フリーハンド**：服の裾・袖口を線で自由になぞって閉じる
- 📏 **直線**：始点と終点をクリックして直線で閉じる
- 🎨 **線の色**：黒・グレー・ネイビー・茶など、キャラの輪郭色に合わせて選択

線で閉じた白い領域は、透過処理時に保護されます。
描いた線で気に入らない部分があれば「↩️ 1本戻す」「🗑️ 全消去」で取り消し可能。

### 2-5. オプション設定

```
☑ サイズ変更（リサイズ）
   → 通常スタンプ（370×320） を選択

☑ 背景透過
   透過モード：
     🛡️ ヒューリスティック方式（白色透過・推奨）
     🎨 スポイト（任意色透過・背景が白以外の時）
     🤖 AI透過（実写向け・ベータ）

☑ メイン画像・タブ画像も生成
```

#### 🎨 スポイト透過の使い方（背景が白以外の時）

4. **許容範囲スライダー** で「似た色も含めるか」調整
5. そのまま「✂️ 切り抜きを実行」
