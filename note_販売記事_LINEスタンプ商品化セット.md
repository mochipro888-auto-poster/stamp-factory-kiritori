# 🆘 困ったときのサポート

### 🔴 白い服が透過で消えてしまう時
→ Step 2-5 の「⚙️ 白服が消えてしまう時の対処」を参照（4つの解決策あり）

### 🔴 テキスト「ごめんね」の文字内側の白が残る時

→ 透過モード3種（ヒューリスティック / スポイト / AI）を試してみてください
→ それでもダメなら **🪄 クリック透過** や **🤍 背景透過を戻す**（ブラシ/矩形/フリーハンド）で個別調整

# 📌 P.S. 隠れた推し機能
